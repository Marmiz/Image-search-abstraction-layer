Image Abstraction Layer
A small project made for Free Code Camp back-end curriculm

What's this?

A simple program where I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

Usage

By visiting api/imagesearch/"your search"You will see a default Json with the 10 most relevant images

api/imagesearch/"your search"

You can paginate through the response by adding an offset

api/imagesearch/"your search"?offset="number"

And you can see the most recent seach history

api/latest/imagesearch
